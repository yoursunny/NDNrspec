NDN Experiment on Fed4FIRE

Instructions
  1. Choose a hardware type in Request RSpec section.
  2. In jFed, select Open Online and paste the request RSpec URI.
  3. Start the experiment, and wait for nodes to become ready.
  4. Export the manifest RSpec, and provide the file in Setup Script section.
  5. Generate a setup script with desired actions.
  6. Paste the script into jFed Multi Command dialog.
  7. Check for script execution status in ESpec tab.
